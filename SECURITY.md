# Security Policy

## Supported versions

Goblin Core is pre-release software. Security fixes are applied to the main branch and the latest published npm versions of each package.

## Reporting a vulnerability

If you discover a security issue:

1. **Do not** open a public issue.
2. Email `security@goblincore.dev` with a detailed report. Include steps to reproduce and any relevant logs.
3. Expect an acknowledgement within 72 hours. We will coordinate a fix and disclosure timeline with you.

Please remember:

- Never send seed phrases, private keys, or wallet exports. Goblin Core is non-custodial and will never ask for them.
- Use encrypted channels when possible.
- Responsible disclosure helps keep the community safe.
